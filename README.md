# Marketplace - Back End

## 📋 Sobre o Projeto

Este projeto foi desenvolvido como parte do MBA Full-stack da Rocketseat e Sirius.

## 🚀 Tecnologias Utilizadas

- NestJS
- TypeScript
- Prisma
- JWT
- Zod
- Vitest

## 🔧 Instalação

1. Clone este repositório
2. Instale as dependências:

```bash
npm install
```

#### API necessário
```bash
git clone https://github.com/DevDanielSantiago/Marketplace---Back-End
```

```bash
# Criar o container
$ docker compose up -d

# Criar a estrutura dos bancos de dados (aplicação e Redis)
$ npx prisma migrate dev
```

## ⚙️ Executando o Projeto

Para iniciar o servidor de desenvolvimento:

Para iniciá-lo:

```bash
# Instalar as dependências
$ npm run start:dev
```

```bash
# Endereço da aplicação
http://localhost:3333.
```

Executar os testes:
```bash
# Iniciar os testes unitários
$ npm run test

# Iniciar os testes end-to-end
$ npm run test:e2e
```

## 📱 Como Usar

Basta realizar um cadastro como usuário e fazer o login.


## 🎯 Funcionalidades

- Cadastro de usuário;
- Atualização dos dados cadastrais;
- Autenticação;
- Cadastro de produto;
- Edição de produto;
- Upload de arquivos;
- Status de produto;
- Categorias;
